# Hello World

A simple Hello World app with a customizable menu for the MicroHydra Cardputer.

## Features
- Displays a "Hello World" screen on startup.
- Press `G0` to bring up a customizable menu.
- Navigate the menu using `;` (UP) and `.` (DOWN).
- Select a menu option using `ENT`.

## Configuration
- Menu items can be customized within the code.
- Colors (background, text, highlight) are customizable.

## Requirements
- Cardputer with the MicroHydra environment.
- MicroHydra libraries (`lib.display`, `lib.userinput`).

## Usage
1. Deploy the app to your Cardputer under `app-source/HelloWorld/`.
2. Launch it through the MicroHydra interface or by automatically executing `hello_world.py`.
